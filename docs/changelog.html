Changelog

Version 1.12.1
April 2026
  • Landscape mode: master volume moved to the header bar as a compact slider, with a drum pad grid replacing the old volume knob area
  • MIDI device connection dialog: redesigned as a two-column layout in landscape for better use of screen space
  • Fixed distortion effect to preserve the original volume when enabled
  • Fixed a crash caused by the Bluetooth permission retry loop on some devices
Version 1.12.0
April 2026
  • Cleaner app icon with smoother wave lines
  • Fixed an issue where the privacy policy and terms of service links could not open on some Android devices
Version 1.11
April 2026
  • MIDI Library: folder import, playlists, landscape layout, and scanning progress dialog — smooth even with large song lists
  • Background audio playback: MIDI music keeps playing after leaving the app, with notification controls
  • 20x faster song switching: instant response when changing tracks
  • Interactive onboarding: guided tour on first launch for new users
  • Per-feature MIDI channel: assign separate channels to keyboard, pitch bend, and modulation
  • Richer recording: pitch bend and modulation events are now captured during recording
  • SoundFont intro: first-time walkthrough when opening the SoundFont manager
  • Resume label shown when paused with a MIDI file loaded; smooth slide transition when switching songs in the library
  • Edge-to-edge support for Android 15+, round adaptive launcher icon
  • Fixed silence after switching SoundFont mid-playback, stuck notes on exit, and intent-based file opening on cold start
  • Eliminated audio stutter caused by UI interactions
