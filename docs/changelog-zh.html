更新日志

版本 1.14.2
2026年4月
  • 音频实验室新增两个独立的缓冲区滑块,分别控制实时演奏与 MIDI 文件播放。可以一边保持演奏低延迟,一边让在较慢机型上的文件播放更稳定、不爆音。
  • Android 音频回调现在通过系统性能提示 API 绑定到大核运行,进一步降低实时演奏时的延迟抖动。
  • MIDI 日志页面在单行内容超出屏幕时,改为整列内容联动横向滚动,不再出现每行各自滑动的错位。
  • 设备连接弹窗的小节标题措辞微调,更清晰一致。
版本 1.14.1
2026年4月
  • 修复连接系统已配对的 BLE MIDI 键盘失败的问题。现在无论设备是通过本应用扫描发现,还是已经被系统蓝牙枚举过的,都能正常连上。
  • 修复 BLE 扫描会持续累积监听器的问题。长时间使用后设备列表刷新频率会异常上升,导致界面卡顿,现已修正。
版本 1.14.0
2026年4月
  • 内置 10 首古典 MIDI 示范曲。首次打开曲库不需要任何准备就能直接试听。
  • 新增四种竖屏舞台布局:鼓垫 + 滚轮 + 键盘、鼓垫 + 双键盘、双鼓垫、滚轮 + 双键盘。随时可在设置 → 按键布局中切换。
  • 新增琴键音名显示方式:只显示 C、全部音名、简谱数字 1-7 (带八度点)。在设置 → 键盘音名中调整。
  • 升级到本版本的用户首次进入主界面会看到一段简短引导,介绍新布局与音名显示功能。
  • 新手引导新增独立的「琴键」与「舞台布局」两个步骤,更好地说明主界面各区域的定位。
  • 修复国际版部分一加设备上 Google Play 订阅支付可能闪退的问题。
  • 修复插入无输入端口的 MIDI 设备时可能崩溃的问题。
  • 修复音色设置左右滑动切换选项卡时偶发的崩溃。
版本 1.13.9
2026年4月

  • 修复导入设置时,单条损坏的记录会导致整次导入失败的问题——现在会跳过坏的条目,其余配置正常导入。
  • 修复录音保存为 MIDI 时偶发的文件选择器卡死。
